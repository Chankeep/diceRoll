# Dice Roll Game
**Create by Cocos Creator 3.8.4**

## 项目概述
这是一个基于 Cocos Creator 3.8.4 开发的骰子滚动游戏。玩家可以同时滚动多个骰子，观察结果，并根据特定规则计算分数。
